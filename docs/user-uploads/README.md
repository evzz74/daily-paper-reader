# 我的上传文献

本目录用于存放用户自己上传的文献。

## 使用方法

1. 点击页面右下角的 **📤 上传按钮**
2. 拖拽或选择要上传的文件（支持 PDF、Markdown、TXT 格式）
3. 填写文献信息（可选）
4. 点击确认上传

## 支持的文件格式

- **PDF** - 学术论文、报告等
- **Markdown (.md)** - 格式化文本
- **TXT** - 纯文本文件

## 文件大小限制

单个文件最大支持 **20MB**

## 数据存储

上传的文献会存储在浏览器的本地存储中，不会自动同步到服务器。
如需备份，请手动导出相关数据。
